================
Mode description
================


* B1 :
    * Left/Rigth : Move the robot from left to right in the world frame
    * Up/Down : Move the robot forward and backward in the world frame

* SPEED : 
    * Up/Down : Increase/Decrease the speed of the robot

* B2 :
    * Left/Right : Rotate J6 of the robot in the gripper frame
    * Up/Down : Move the robot up and down in the world frame

* DRINK : 
    * Left/Right : Rotate J6 and and move the robot up and down to help with drinking motion
    * Up/Down : Move the robot up and down in the world frame

* B3 : 
    * Left/Right : Rotate on yaw the robot in the gripper frame
    * Up/Down : Rotate on pitch the robot in the gripper frame

* B4 : 
    * Up/Down : Open/Close the gripper

* DEPLOY :
    * Up/Down : Deploy/Retract the robot arm
